# @TypeScript Log Decorator
A Node.js module that allow fast logging

## Installation
```sh
npm install ts-log-decorator --D
```

## Usage

### Javascript

```javascript
var logMethod = require('log-decorator');
```

### TypeScript

```typescript
import { logMethod } from 'log-decorator';
console.log(getPlural('Goose'))
```

## Test

```sh
npm run test
```
